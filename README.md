# RAPPI-DITTO
servicio de entregas a domicilios v
Descripción

Este proyecto consiste en una página web desarrollada únicamente con HTML5. Su objetivo es representar la estructura básica de una aplicación de domicilios, donde los usuarios pueden visualizar productos, acceder al carrito de compras y completar la información necesaria para realizar un pedido.

Estructura de la página

La página está organizada en las siguientes secciones:

- Encabezado (Header):
  
  - Nombre de la aplicación (EcoDelivery).
  - Barra de búsqueda.
  - Menú de navegación con botones de Inicio, Restaurantes, Promociones y Carrito.

- Banner principal:
  
  - Presenta un mensaje de bienvenida y una breve descripción del servicio.

- Categorías de productos:
  
  - Hamburguesas.
  - Pizzas.
  - Sushi.
  - Bebidas.
  - Postres.
  
  Cada producto contiene:
  
  - Nombre.
  - Precio.
  - Botón "Agregar".

- Carrito de compras:
  
  - Espacio donde se mostrarán los productos agregados.
  - Total de la compra.
  - Botón para confirmar el pedido.

- Inicio de sesión:
  
  - Campo para usuario.
  - Campo para contraseña.
  - Botón para iniciar sesión.

- Datos del domicilio:
  
  - Nombre del cliente.
  - Dirección.
  - Teléfono.

- Estado del pedido:
  
  - Sección destinada a mostrar información sobre el proceso del pedido.


Objetivo

El propósito de este proyecto es diseñar la estructura de una página web para un servicio de domicilios utilizando únicamente HTML, organizando correctamente el contenido y preparando la base para futuras mejoras con otros lenguajes de desarrollo web
